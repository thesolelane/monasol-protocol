# NexusBridge Backend Setup

## Step 1 — Supabase

1. Go to https://supabase.com and create a free project
2. Once created, go to **SQL Editor** in the left sidebar
3. Paste the entire contents of `schema.sql` and click **Run**
4. Go to **Project Settings → API**
5. Copy your **Project URL** and **anon/public key**

## Step 2 — Replit Secrets

In your Replit project, click the **Secrets** tab (lock icon) and add:

```
SUPABASE_URL      = https://your-project-ref.supabase.co
SUPABASE_ANON_KEY = your-anon-key-here
VITE_API_URL      = http://localhost:3001
PORT              = 3001
```

## Step 3 — Install backend dependencies

In your Replit shell, run:

```bash
npm install express cors @supabase/supabase-js
```

## Step 4 — Add server.js to your project

Copy `server.js` to your Replit project root.

If your Replit project uses a `package.json`, add this script:

```json
{
  "scripts": {
    "dev": "node server.js",
    "start": "node server.js"
  }
}
```

## Step 5 — Add MoveInModal to your frontend

Copy `MoveInModal.tsx` into your `src/components/` folder.

Then update `home.tsx` to import and use it:

```tsx
import { MoveInModal } from "@/components/MoveInModal";

// Add state:
const [isMoveInOpen, setIsMoveInOpen] = useState(false);

// Replace or add alongside the existing RentVaultModal:
<Button onClick={() => setIsMoveInOpen(true)}>
  Move In
</Button>

<MoveInModal
  isOpen={isMoveInOpen}
  onClose={() => setIsMoveInOpen(false)}
  onSuccess={(vault) => console.log('Moved in:', vault)}
  connectedWallet={solanaWalletAddress}  // pass your connected wallet address
  availableNfts={[]}  // pass real NFTs once wallet is connected
/>
```

## Step 6 — Test it

1. Start the backend: `node server.js`
2. Open your Replit frontend
3. Click Move In
4. The modal walks through: select NFT → set deposit → review fees → confirm
5. Check your Supabase table — a vault row should appear

## API Endpoints

| Method | Route | Description |
|---|---|---|
| GET | /health | Server health check |
| GET | /lockers | List all active lockers |
| GET | /vault/:wallet | Get all vaults for a wallet |
| GET | /vault/ref/:ref | Get vault by abbreviated ref |
| POST | /move-in | Create vault, register NFT, collect lease |
| POST | /deposit | Record a deposit into existing vault |
| PATCH | /vault/:id/security-mode | Update circuit breaker mode |

## Fee structure (hardcoded, no asset value reference)

| Fee | Amount | When |
|---|---|---|
| Lifetime lease | 0.05 SOL | Once, at vault creation |
| Move-in fee | 0.001 SOL | Once, at vault creation |
| Deposit fee | 0.001 SOL | Each deposit |
| Withdrawal fee | 0.001 SOL | Each withdrawal |

All fees are flat. None reference vault contents or transaction value.
